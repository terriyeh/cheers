Design Specifications

  • Background Dimensions: 800×600px (4:3 aspect ratio)
  • File Format: WebP (~40KB per image)
  • Visual Style: Soft pixel art with painterly textures
  • Pet Size: Fixed at 64×64px (does not scale)
  • Pet Ground Level: 70% from top
  • Horizon Line: 45% from top
  • Movement: Edge-to-edge horizontal (CSS animation)
  • Layers: Background (z:0) → Pet (z:5) → Heart (z:10)
